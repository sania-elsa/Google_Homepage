# Google Homepage Clone

This project is a front-end clone of the Google homepage built using HTML, CSS, and JavaScript. It replicates the layout and basic functionalities of the Google search page.

## Features
- Clean and responsive Google-like UI
- Functional search input
- Search history using Local Storage
- Voice search (Web Speech API)
- Image upload simulation
- Interactive search dropdown

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Project Structure
- index.html
- style.css
- script.js

## Purpose
This project was created as part of an assignment to practice front-end development concepts including layout design, DOM manipulation, and browser APIs.
